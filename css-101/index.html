The basic language of the web HTML

Posted by Laura Jones on June 21st 2023

All modern websites and web applications are built using three fundamental technologies,HTML,CSS and Javascript.These are languages of the web

In this post,we will focus on HTML.What HTML is all about and and why you too should learn about it.

What is HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.

In HTML each element is made up of 3 parts

  1. The opening tag
  2. The closing tag
  3. The actual element

HTML consists of a series of elements. HTML elements tell the browser how to display the content.

Why should you learn HTML?

Here are some of the reasons why you should learn the fundamental language of the web

  • HTML fosters creativity.
  • HTML forms the foundation of websites
  • HTML improves problem-solving abilities.
  • HTML is fun.
  • HTML leads to complex programming languages.
